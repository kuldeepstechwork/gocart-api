Production-ready E-commerce REST API built with Go
GoCart is a scalable and modular e-commerce backend built using Go, following clean architecture and industry best practices. It provides RESTful APIs for core e-commerce functionalities such as authentication, product management, cart, and order processing.
This project is designed as a real-world backend system, not a toy example, and is suitable for learning, interviews, and portfolio demonstration.
